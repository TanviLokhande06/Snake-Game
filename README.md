# Snake Game 🐍

A simple and fun **Snake Game** built using **HTML, CSS, and JavaScript**.
Control the snake, eat food, grow longer, and try to beat your high score!

---

## 🎮 Features

* Classic Snake gameplay
* Score tracking system
* High score saved using `localStorage`
* Game timer
* Start and Restart functionality
* Responsive grid-based board
* Keyboard controls using arrow keys
* Game Over modal

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/tanvilokhande06/snake-game.git
```

### 2. Navigate to the Project Folder

```bash
cd Snake Game
```

### 3. Open the Game

Simply open the `index.html` file in your browser.

---

## 🎯 How to Play

* Press the **Start** button to begin the game.
* Use the keyboard arrow keys to move the snake:

| Key            | Action     |
| -------------- | ---------- |
| ⬆️ Arrow Up    | Move Up    |
| ⬇️ Arrow Down  | Move Down  |
| ⬅️ Arrow Left  | Move Left  |
| ➡️ Arrow Right | Move Right |

* Eat the food to grow the snake and increase your score.
* Avoid hitting the walls.
* Try to beat your highest score!
